🌦 WeatherWeb
A modern weather dashboard that shows real-time weather information, forecasts, humidity, and wind speed using the Open-Meteo API.

🚀 Features :
🌍 Search weather by city
🌡 Real-time temperature display
💧 Humidity information
🌬 Wind speed data
☁ Weather condition display
📅 5-Day weather forecast
🕘 Recent search history
🌙 Light/Dark mode UI
 
🛠 Tech Stack
This project is built using:
HTML
CSS
JavaScript
Open-Meteo API

🌐 API Used
This project uses the Open-Meteo Weather API for real-time weather data.
Website:
https://open-meteo.com/
